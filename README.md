# ThreatDetector

Project scaffold for a transaction threat detection app.

## Structure

- `app.py` - Flask entry point
- `database.py` - SQLite setup helper
- `monitor.py` - Monitoring logic placeholder
- `generator.py` - Sample data generation placeholder
- `detector.py` - Detection logic placeholder
- `templates/index.html` - Main page template
- `static/css/style.css` - Stylesheet
- `static/js/script.js` - Client-side script
- `database/transactions.db` - SQLite database file
